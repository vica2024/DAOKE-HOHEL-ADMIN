# 民宿小程序+H5

#### 介绍
2020年到2022年，这两年间的疫情对文旅行业带来沉重的打击，作为一位热爱旅游的互联网技术小菜鸟，也想为疫情过后旅游行业略尽绵薄之力，希望旅游行业尽快复苏，然后我好出去游山玩水，在疫情期间有很长一段时间宅在家里，因此潜心搞研发，现在这款民宿小程序+H5就是在下给广大民宿主一份小礼物。
