## 安装使用

- 获取代码
```
git clone https://github.com/vica2024/DaokeHomestay-vue.git
```
- 安装依赖
```
cd DaokeHomestay && yarn install 
```
或者
```
cd DaokeHomestay && npm install
```
- 运行项目
```
yarn dev
```
或者
```
nup run dev
```
- 打包发布
```
yarn build
```
