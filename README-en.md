## Install Guide

- pull the code
```
git clone https://github.com/vica2024/DaokeHomestay.git
```
- Install Dependencies
```
cd DaokeHomestay && yarn install 
```
or
```
cd DaokeHomestay && npm install
```
- run project
```
yarn dev
```
or
```
nup run dev
```
- build the project to launch
```
yarn build
```
